# Multi-Platform Matrix Build Demo

This repository demonstrates a matrix CI strategy in GitHub Actions that builds and uploads artifacts for multiple Python versions.

## Workflow Features
- Matrix build: Python 3.9, 3.10, 3.11
- Each job uploads a uniquely-named artifact
- Artifacts prefixed as build-00750ec-...
- Actions identifier: matrix-00750ec appears in workflow logs
